## 🚀 Deployment Instructions

### Local Testing
```bash
cd agent-challenge
npm install
npm run build
npm run start
# Access at http://localhost:8080
```

### Docker Deployment
```bash
docker build -t supply-chain-agent .
docker run -p 8080:8080 supply-chain-agent
```

### Nosana Deployment
```bash
npm run deploy:agent
```
